## Getting started

1. Go to [brew.sh](https://brew.sh/) and copy the command to install Homebrew. Paste on **Terminal**.
2. Install wezterm `brew install --cask wezterm`

### Git Setup

```sh
# GitHub CLI
brew install gh
gh auth login

# Git Setup (Use your Github username + email)
git config --global user.name "anvllmr"
git config --global user.email "your.email@example.com"

# (optional) Do git even faster
brew install lazygit
lazygit4
```

## Apps!

- Zed - `brew install --cask zed`
- Opencode - `brew install anomalyco/tap/opencode`
- Notion - `brew install --cask notion`
- Raycast - `brew install --cask raycast`
  - Replace capslock with "Hyper Key ✨" (Go to Advanced, Hyper Key = Capslock)
 - Shortcuts:
   - ✨ + N - Notion
   - ✨ + 2 - Brave
   - ✨ + 3 - Chrome
   - ✨ + S - Slack
   - ✨ + Z - Zed
   - ✨ + W - Wezterm
